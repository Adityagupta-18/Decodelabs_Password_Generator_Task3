# Python Password Generator

A simple Python-based Password Generator developed as part of the DecodeLabs Industrial Training Program. This project generates secure and random passwords based on a user-defined length using alphabets, numbers, and special characters.

## Features

* User-defined password length
* Random password generation
* Includes:
  * Uppercase and lowercase alphabets
  * Digits
  * Special characters
* Character shuffling for improved randomness
* Function-based implementation
* Repeat generation using loops

## Technologies Used

* Python 3

## Concepts Covered

* Functions
* Loops
* Conditional Statements
* Random Module
* String Handling
* Lists
* User Input Handling

## Project Structure

password_generator.py    -> Main Python program

## How It Works

1. User enters desired password length
2. Program generates:
   * Half characters as alphabets
   * One-fourth as digits
   * One-fourth as special characters
3. All characters are shuffled randomly
4. Final secure password is displayed
5. User can generate passwords repeatedly

## Example Output
Enter password length: 12

Generated Password:
aK9@xP2#Lm8!
